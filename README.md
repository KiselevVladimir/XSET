# Тестирование производительности WebTours с LoadRunner

## Структура проекта

```
├── Документация/     # Тестовая документация, планы тестирования
├── Результаты/       # Отчеты, графики, результаты тестов
├── Скрипты/         # LoadRunner скрипты (.usr файлы)
├── Сценарии/        # Файлы сценариев тестирования (.lrs)
└── README.md        # Описание проекта
```

## Цели тестирования
- Нагрузочное тестирование приложения WebTours
- Анализ производительности под разной нагрузкой
- Определение "узких мест" системы

## Используемые технологии
- **Micro Focus LoadRunner** - инструмент нагрузочного тестирования
- **WebTours** - тестовое приложение
- **Протокол**: Web (HTTP/HTML)

## Начало работы
1. Запустить WebTours сервер: `StartServer.bat`
2. Открыть приложение: http://localhost:1080/WebTours
3. Использовать учетные данные: jojo/bean
